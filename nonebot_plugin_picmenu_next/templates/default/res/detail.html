nonebot-plugin-bilichat

By djkcyl & Well404 | v6.2.7

简介

全功能的 bilibili 内容解析器及订阅器

用法

视频、专栏、动态解析直接发送链接、小程序、xml卡片即可, 指令请参考 https://github.com/Well2333/nonebot-plugin-bilichat

功能

No.1

添加订阅

触发条件：群聊 + 主人
触发方式：/sub
UP 主的昵称或 UID

No.2

移除订阅

触发条件：群聊 + 主人
触发方式：/unsub
UP 主的昵称或 UID

No.3

查看本群订阅

触发条件：群聊 + 无限制
触发方式：/check
无

No.4

设置是否 at 全体成员, 仅 OB11 有效

触发条件：群聊 + 主人
触发方式：/atall
UP 主的昵称或 UID, 或 `全局`

No.5

验证码登录

触发条件：无限制 + 主人
触发方式：/smslogin
无

No.6

二维码登录

触发条件：无限制 + 主人
触发方式：/qrlogin
无
